# Employee-Management-System | HTML, C#

## Features:

. Admin Login with session management

. Employee Management - Add, edit, delete employees

. Dashboard with visual charts (Project Status & Department Stats)

. Project Management - Add, edit, delete projects

. Leave Management - Submit and track leave requests

. Salary Management - View employee salaries

. DataTables Integration - Sortable, searchable tables

. Responsive UI with Bootstrap 5

. Inline Editing - Edit records directly in tables

. Soft & Hard Delete options

## Tech Stack:

. ASP.NET Core MVC (.NET 6/7/8)

. Entity Framework Core (Database First)

. SQL Server

. Bootstrap 5

. jQuery & DataTables

. Chart.js for analytics

. FontAwesome icons

## How to Use: 

. Login with admin credentials (email: admin, password: admin123)

. Dashboard - View employee count, project count, and analytics charts

. Employee List - View all employees with inline edit/delete options

. Add Employee - Fill the form to add new employees

. Add Project - Assign projects to employees

. Projects List - View all projects with edit/delete options

. Add Leave - Submit leave requests for employees

## Stored Procedures Used:

. GetEmployeeSalaryDetails - Fetch employees with salary info

. GetEmployeeProjects - Fetch projects with employee names
